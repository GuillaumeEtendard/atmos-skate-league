# Atmos Skate League

Site web de l'Atmos Skate League — la 1ère ligue de rollers par AtmosGear.

## Développement local

Prérequis : Node.js et npm — [installer avec nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

```sh
# Cloner le dépôt
git clone <URL_DU_DEPOT>

# Entrer dans le projet
cd atmos-skate-league

# Installer les dépendances
npm i

# Lancer le serveur de développement
npm run dev
```

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Déploiement

Le projet peut être déployé sur Vercel, Netlify ou tout hébergeur supportant les applications Vite/React (build : `npm run build`).
